# Agent Workspace

This folder provides lightweight navigation aids for automated agents working in this repo.

## Structure
- `subagents/repo-map.md` — quick map of major folders and what they contain.
- `subagents/workflow-checklist.md` — suggested step-by-step flow for understanding and editing the project safely.

## Usage
1. Read `subagents/repo-map.md` before making changes.
2. Follow `subagents/workflow-checklist.md` for repo exploration and validation.
3. Keep docs updated if major project structure changes.
