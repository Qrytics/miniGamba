# miniGamba - Assets

This directory contains all static assets for miniGamba.

## Structure

```
assets/
├── images/              # Image assets
│   ├── icons/          # App and system tray icons
│   ├── games/          # Game-specific images
│   │   ├── slot-symbols/
│   │   ├── cards/
│   │   ├── dice/
│   │   └── misc/
│   ├── backgrounds/    # Background images
│   └── cosmetics/      # Cosmetic item images
├── sounds/             # Sound effects
│   ├── ui/            # UI sounds
│   ├── games/         # Game sounds
│   │   ├── slot-machine/
│   │   ├── cards/
│   │   └── coin-flip/
│   └── music/         # Background music
└── fonts/             # Custom fonts
```
